# 📸 Advanced Image Gallery Website

A modern and responsive Image Gallery Website built using HTML, CSS, and JavaScript.

This project includes multiple gallery categories, fullscreen lightbox view, image filters, favorites system, dark mode, keyboard navigation, responsive design, and many advanced gallery features.

✅ Responsive Image Gallery
